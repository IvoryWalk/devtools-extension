## ✨ 特性

- 🚀 **零依赖**：纯原生JavaScript开发，无任何第三方依赖
- 🔒 **安全隐私**：所有数据本地存储，不上传任何服务器，Manifest V3规范
- 🎨 **精致UI**：卡片式功能区、Google四色高频标记、可拖拽分割线、输出区行号
- 🌗 **主题三态**：跟随系统 / 深色 / 浅色，一键切换
- 🔤 **全局字号**：7级字号等比例缩放（极小→极大），适应不同屏幕和视力需求
- ⚡ **即开即用**：新标签页打开，打开即进入文本比对编辑模式，左右双栏可编辑
- 💾 **数据持久化**：配置和历史记录自动保存，支持导入导出备份
- 🎯 **智能排序**：根据使用频率自动排序常用功能，Top-4按钮Google色边框标记
- 📦 **数据暂存**：内置剪贴板暂存功能，支持重命名，方便复制粘贴常用内容

## 🛠️ 功能列表

> 默认顺序：JSON → SQL → 时间戳 → 文本比对 → 编解码 → Cron

### 📋 JSON
- JSON格式化/美化
- JSON压缩/最小化
- JSON转义/去转义
- JSON Key排序
- JSON格式验证
- 递归提取指定Key
- 递归排除指定Key

### 🗄️ SQL
- **SQL美化（格式化）**：智能识别 DML / DDL 语句，自动选择最佳排版风格
  - DML 风格：Tab缩进、AND/OR前置换行、SELECT字段逗号前置、CASE WHEN展开、ORDER BY/GROUP BY内联
  - DDL 风格：数据类型保持小写、列定义逐行、USING BTREE前置、表选项规范化（DEFAULT CHARSET→CHARSET、COMMENT=→COMMENT）
  - 嵌套查询括号独立成行并对齐上层语句
  - 括号内逗号紧凑不换行（VALUES/IN/列定义）
- **SQL净化**：关键字大写、运算符空格规整、移除所有注释（`--`/`#`/`/* */`）、单行输出
- **语法高亮**：关键字（蓝）、字符串（绿）、注释（灰斜体）、数字（橙）、函数名（粉），适配明暗双主题
- 零依赖纯正则实现，不改变SQL语义

### ⏰ 时间戳
- 获取当前时间戳
- 时间戳转日期时间
- 日期时间转时间戳
- 支持秒/毫秒自动识别

### 🔬 文本比对
- **行比对**：逐行对齐，删除行红底+左红线，新增行绿底+左绿线
- **词比对**：词级别内联高亮，删除文本红色+删除线，新增文本绿色
- **字符比对**：字符级别内联高亮，精确定位差异位置
- **双栏编辑模式**：打开插件即进入编辑态，左栏原始文本、右栏修改文本，均可直接编辑
- **统计栏**：显示新增/删除/相同行数或词数，一目了然
- **重新编辑**：一键回到编辑模式，文本内容保留
- 零依赖自实现 Myers O(ND) diff 算法

### 🔐 编解码
- URL编码/解码
- Unicode编码/解码
- GBK编码/解码（浏览器原生支持，零体积）
- Base64编码/解码

### ⏱️ Cron表达式解析
- Linux Cron（5位：分 时 日 月 周）
- Spring Cron（6位：秒 分 时 日 月 周）
- Quartz Cron（6-7位：秒 分 时 日 月 周 年）
- 支持`?`、`L`（每月最后一天）等特殊字符
- 自动计算未来N次执行时间

### 📦 数据暂存
- 自动保存输入/输出内容
- 支持重命名暂存项
- 可配置最大条数和容量
- 抽屉式面板，不占主界面空间

### ⚙️ 个性化配置
- 功能行拖拽排序
- 每个功能按钮独立显示/隐藏开关
- 按钮顺序弹窗调整，支持置顶/上移/下移/置底
- 自动按使用频率排序，阈值可配置
- Top-4高频按钮Google色边框标记（红/蓝/黄/绿）
- 全局字号7级调节（极小/较小/略小/中等/略大/较大/极大），等比例缩放
- 主题三态切换（跟随系统/深色/浅色）
- 数据导入导出，方便备份迁移
- 使用统计重置，恢复默认设置

## 📦 安装方法

### 方式一：源码安装（推荐）
1. 下载本项目源码到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击左上角「加载已解压的扩展程序」
5. 选择本项目文件夹即可完成安装
6. 点击工具栏的DevToolBox图标，即可在新标签页打开使用

### 方式二：CRX安装
1. 下载发布页的 `.crx` 文件
2. 打开 `chrome://extensions/`，将crx文件拖入页面即可安装

## 🖥️ 使用说明

### 快捷键
- `Ctrl/Cmd + Enter`：快速格式化JSON
- `Esc`：关闭暂存面板/弹窗

### 界面布局
- **功能区**：卡片式布局，每个功能块有蓝色细边框，左侧logo+描述，右侧按钮按内容自然宽度排列
- **输入/结果区**：中间可拖拽分割线，动态调整两区域宽度比例，比例自动保存
- **行号显示**：输入区和结果区均显示行号，滚动联动
- **窄屏适配**：屏幕较窄时自动变为上下结构，页面可滚动

### 基本使用
1. 在左侧输入框粘贴需要处理的内容
2. 点击对应功能卡片中的按钮即可处理
3. 结果显示在右侧输出框（带行号和语法高亮）
4. 点击「复制」按钮一键复制结果，「保存」存入暂存，「清空」重置内容
5. 拖拽输入/结果区中间的分割线可调整两区域宽度

### 文本比对使用
1. **打开插件即进入文本比对编辑模式**：左栏「原始文本」、右栏「修改文本」，均为可编辑 textarea
2. 在左右两栏分别输入待比对的文本
3. 点击功能区的「行」/「词」/「字符」按钮执行对应粒度的比对
4. 比对结果以双栏 diff 视图呈现（只读），顶部显示统计栏（新增/删除/相同数量）
5. 点击 panel 标题栏的「重新编辑」按钮回到编辑模式，文本内容保留
6. 点击任意非文本比对功能按钮（如 JSON 格式化）自动退出比对模式，恢复正常输入/结果界面

### SQL美化示例

**输入：**
```sql
select id,name,age from user where age > 18 and status = 1 order by create_time desc limit 10
```

**美化输出：**
```sql
SELECT id
	, name
	, age
FROM user
WHERE age > 18
	AND status = 1
ORDER BY create_time DESC
LIMIT 10
```

### 自定义排序
1. 点击右上角「设置」进入配置页面
2. 关闭「按使用频率自动排序」开关
3. 拖拽功能行调整整体顺序
4. 点击每行的「按钮排序」，在弹窗中调整该行按钮顺序和显示开关
5. 点击「保存设置」生效

### 全局字号设置
1. 进入设置页面，找到「全局字号」区块
2. 选择7个等级之一：极小 / 较小 / 略小 / 中等（默认）/ 略大 / 较大 / 极大
3. 选择时实时预览效果（整页等比例缩放）
4. 点击「保存设置」持久化到本地

### 主题切换
- 点击工具栏右侧主题按钮，三态循环切换：跟随系统 → 深色 → 浅色 → 跟随系统
- 「跟随系统」模式自动跟随操作系统的明暗设置

### 数据备份
- **导出**：设置页面点击「导出数据」，自动下载JSON备份文件
- **导入**：选择备份文件，可选择合并或覆盖现有数据

## 📁 项目结构
```
devtools-extension/
├── manifest.json          # 扩展配置文件（Manifest V3）
├── background.js          # 后台服务，点击图标打开新标签页
├── index.html             # 主页面
├── settings.html          # 设置页面
├── css/
│   └── style.css          # 样式文件，支持明暗主题
├── js/
│   ├── app.js             # 主应用逻辑
│   ├── settings.js        # 设置页面逻辑
│   └── modules/
│       ├── json-tools.js  # JSON处理模块
│       ├── sql-tools.js   # SQL美化/净化模块
│       ├── diff-tools.js  # 文本比对模块（Myers diff算法）
│       ├── codec-tools.js # 编解码模块
│       ├── timestamp.js   # 时间戳模块
│       ├── cron-parser.js # Cron解析模块
│       ├── storage.js     # 本地存储模块
│       └── usage-stats.js # 使用统计模块
├── icons/                 # 扩展图标（16/48/128/640px）
├── test-sql.js            # SQL工具测试脚本（25用例，node运行）
└── README.md              # 说明文档
```

- 不收集任何用户信息，不联网，无广告，无追踪
- 源码完全开放，可自行审计
