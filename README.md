## ✨ 特性

- 🚀 **零依赖**：纯原生JavaScript开发，无任何第三方依赖，体积仅38KB
- 🔒 **安全隐私**：所有数据本地存储，不上传任何服务器，Manifest V3规范
- 🎨 **美观简洁**：支持明暗主题自动切换，macOS风格设计
- ⚡ **即开即用**：新标签页打开，无需切换面板，所有功能平铺展示
- 💾 **数据持久化**：配置和历史记录自动保存，支持导入导出备份
- 🎯 **智能排序**：根据使用频率自动排序常用功能，也可手动自定义顺序
- 📦 **数据暂存**：内置剪贴板暂存功能，支持重命名，方便复制粘贴常用内容

## 🛠️ 功能列表

### 📋 JSON处理
- JSON格式化/美化
- JSON压缩/最小化
- JSON转义/去转义
- JSON Key排序
- JSON格式验证
- 递归提取指定Key
- 递归排除指定Key

### 🔐 编解码
- URL编码/解码
- Unicode编码/解码
- GBK编码/解码（浏览器原生支持，零体积）
- Base64编码/解码

### ⏰ 时间戳转换
- 获取当前时间戳
- 时间戳转日期时间
- 日期时间转时间戳
- 支持秒/毫秒自动识别

### ⏱️ Cron表达式解析
- Linux Cron（5位：分 时 日 月 周）
- Spring Cron（6位：秒 分 时 日 月 周）
- Quartz Cron（6-7位：秒 分 时 日 月 周 年）
- 支持`?`、`L`（每月最后一天）等特殊字符
- 自动计算未来N次执行时间

### 📦 数据暂存
- 自动保存输入/输出内容
- 支持重命名暂存项
- 可配置最大条数和容量
- 抽屉式面板，不占主界面空间

### ⚙️ 个性化配置
- 功能行拖拽排序
- 每个功能按钮独立显示/隐藏开关
- 按钮顺序弹窗调整，支持置顶/上移/下移/置底
- 自动按使用频率排序，阈值可配置
- 数据导入导出，方便备份迁移
- 使用统计重置，恢复默认设置

## 📦 安装方法

### 方式一：源码安装（推荐）
1. 下载本项目源码到本地
2. 打开Chrome浏览器，进入 `chrome://extensions/`
3. 开启右上角「开发者模式」
4. 点击左上角「加载已解压的扩展程序」
5. 选择本项目文件夹即可完成安装
6. 点击工具栏的DevToolBox图标，即可在新标签页打开使用

### 方式二：CRX安装
1. 下载发布页的 `.crx` 文件
2. 打开 `chrome://extensions/`，将crx文件拖入页面即可安装

## 🖥️ 使用说明

### 快捷键
- `Ctrl/Cmd + Enter`：快速格式化JSON
- `Esc`：关闭暂存面板/弹窗

### 基本使用
1. 在左侧输入框粘贴需要处理的内容
2. 点击对应功能按钮即可处理
3. 结果显示在右侧输出框
4. 点击「复制」按钮一键复制结果，「保存」存入暂存，「清空」重置内容

### 自定义排序
1. 点击右上角「设置」进入配置页面
2. 关闭「按使用频率自动排序」开关
3. 拖拽功能行调整整体顺序
4. 点击每行的「按钮排序」，在弹窗中调整该行按钮顺序和显示开关
5. 点击「保存设置」生效

### 数据备份
- **导出**：设置页面点击「导出数据」，自动下载JSON备份文件
- **导入**：选择备份文件，可选择合并或覆盖现有数据

```
├── manifest.json          # 扩展配置文件（Manifest V3）
├── background.js          # 后台服务，点击图标打开新标签页
├── index.html             # 主页面
├── settings.html          # 设置页面
├── css/
│   └── style.css          # 样式文件，支持明暗主题
├── js/
│   ├── app.js             # 主应用逻辑
│   ├── settings.js        # 设置页面逻辑
│   └── modules/
│       ├── json-tools.js  # JSON处理模块
│       ├── codec-tools.js # 编解码模块
│       ├── timestamp.js   # 时间戳模块
│       ├── cron-parser.js # Cron解析模块
│       ├── storage.js     # 本地存储模块
│       └── usage-stats.js # 使用统计模块
├── icons/                 # 扩展图标（16/48/128/640px）
└── README.md              # 说明文档
```

## 🔒 隐私说明

- 本扩展**完全本地运行**，不向任何服务器发送数据
- 所有配置、暂存内容、使用统计都保存在浏览器本地`chrome.storage.local`中
- 不收集任何用户信息，不联网，无广告，无追踪
- 源码完全开放，可自行审计
