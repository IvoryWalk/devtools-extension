# Chrome 开发者工具插件 - 开发进度管理

## 项目概述
一个功能丰富的Chrome浏览器扩展，为开发者提供常用的编码、格式化、转换等工具集合。

## 功能清单
- [ ] 页面布局框架（左右分栏+顶部功能导航）
- [ ] JSON工具模块
  - [ ] JSON格式化（美化缩进）
  - [ ] JSON压缩（去除空白）
  - [ ] 字符串转义
  - [ ] 字符串去除转义
  - [ ] Key按字母排序
  - [ ] 语法关键字高亮
- [ ] 编解码模块
  - [ ] URL编码/解码
  - [ ] Unicode编码/解码
  - [ ] GBK编码/解码
  - [ ] Base64编码/解码
- [ ] 时间戳转换模块
  - [ ] 获取当前时间戳（秒/毫秒）
  - [ ] 时间戳转易读格式
  - [ ] 时间戳转数据库格式（YYYY-MM-DD HH:mm:ss）
  - [ ] 日期格式转时间戳
- [ ] 数据暂存模块
  - [ ] 本地存储（最多50条）
  - [ ] 自动生成摘要
  - [ ] 列表搜索功能
  - [ ] 一键复制/删除
- [ ] Cron表达式解析
  - [ ] Cron语法校验
  - [ ] 计算最近5次执行时间

## 项目结构
```
devtools-extension/
├── manifest.json          # Chrome扩展配置文件
├── popup.html             # 弹出页面（点击图标打开）
├── PROGRESS.md            # 本文件，进度管理
├── README.md              # 使用说明
├── css/
│   └── style.css          # 全局样式
├── js/
│   ├── app.js             # 主应用入口
│   └── modules/           # 功能模块
│       ├── json-tools.js      # JSON工具
│       ├── codec-tools.js     # 编解码工具
│       ├── timestamp.js       # 时间戳转换
│       ├── storage.js         # 数据暂存
│       └── cron-parser.js     # Cron解析
├── icons/                 # 扩展图标
└── popup/                 # 弹出页相关资源
```
